# ELLY Website

Website tinh cho dich vu phun xam tham my tai nha ELLY.

## Them Bai Moi Vao Muc Kien Thuc

Khi nguoi dung yeu cau:

```text
Them bai moi: [ten chu de]
```

Codex can lam theo:

```text
SEO-ARTICLE-GUIDE.md
kien-thuc/template-bai-viet-seo.html
```

Quy tac quan trong:

- Khong tao lai website tu dau.
- Khong xoa file cu.
- Khong doi giao dien chinh.
- Giu ten cac file HTML dang co.
- Tao bai moi trong thu muc `kien-thuc/`.
- Cap nhat `kien-thuc.html`, `sitemap.xml`, va cac bai lien quan.
- Khong them `kien-thuc/template-bai-viet-seo.html` vao danh sach bai hoac sitemap.
